# Mintlify Landing Page Clone 

This project is a **frontend clone of the Mintlify Landing Page**, recreated to practice modern UI development, layout structuring, and responsive web design.

The goal of this project was to replicate the clean developer-focused interface used by Mintlify's official website.

---

## Sections Recreated

The following sections from the original Mintlify landing page were recreated:

- **Navigation Bar**
  - Logo
  - Navigation links
  - Call-to-action buttons

- **Hero Section**
  - Main headline
  - Product description
  - Primary CTA buttons
  - Minimal modern layout

- **Trusted Companies / Social Proof Section**
  - Brand logos
  - Clean alignment styling

- **Features Section**
  - Feature cards
  - Icons and descriptions
  - Grid-based layout

- **Product Showcase**
  - Documentation interface preview
  - Visual UI representation

- **Call To Action Section**
  - Engagement-focused design
  - Conversion-style layout

- **Footer**
  - Links
  - Product navigation
  - Branding

---

## Fonts Used

Inspired by Mintlify's modern developer aesthetic:

- **Primary Font:** Inter
- **Fallback:** system-ui, sans-serif

These fonts provide readability and a clean documentation-style appearance.
